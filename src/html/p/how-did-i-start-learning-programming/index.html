我是如何开始学习编程的

我会尽量回答这个问题。

编程是一个非常有趣的方向，已经收到 令人难以置信的名气在2020年，仍然有名望。 在更大程度上这 部分原因是由于程序员的巨大需求，他们得到了一切 较小。 我们稍后再谈远程工作。

第一个项目

我在肩上开发的第一个项目，我的 主管帮助-游戏项目Babka在狩猎。 这是一个 不同版本的游戏，呈现在谷歌播放。

关于这个游戏我 写道 一个播客，我告诉了我是如何开发这个游戏的。 我最终得到了什么。

我从C语言开始学习编程。 说实话，它 这是非常困难的，因为C编程语言甚至 经过四年的积极发展并不那么容易，而且我不是 我说的是从给定的开始学习编程 语言。

为了更好地理解编程的意义与我的 科学负责人决定开发自己的游戏项目，与 我在比赛中表演并参加了比赛。

我的第一个网站

除了有一个游戏项目在开发中，我一直有 有趣的是开发自己的网站，人们会使用。 和 我我也对这个方向产生了兴趣。

和我的主管一起，我们建立了服务器，我坐下来为 发展。 第一个网站远非web中的理想示例 开发，这就是为什么我很快用其他解决方案替换它, 我还没有达到WordPress CMS。

在我的网站上实现CMS WordPress后，我有点沉入 网站开发和我不太喜欢它。 我真的很想 领导他们的网站是独立的发展，我能够 学习Bootstrap框架后实现。

所以我开始领导这个项目的开发live.creagoo.ru ，关于 我之前写过一篇文章。 已经在这一点上，我是积极的 从事游戏项目和网络资源的开发.

其他课程

我还积极地用Python编写代码，主要是他创建的 算法和从来没有工作在他的系统的发展，因为 我一直使用其他编程语言来解决他们的问题。 技术问题。

在Python中，我编写了在控制台中工作的程序。 创建的web 资源与用户授权和解决问题与人工 情报。

我是怎么知道编程的

我的童年都与科技联系在一起：电话、笔记本电脑和 其他装置。 我一直被这个方向所吸引，我喜欢这个方向 是有趣的理解和修复工艺品。

我的思想使我能够用乐高和乐高创造出令人难以置信的东西。 其他建筑材料。 因此，决定发送 我到一个3D建模圈，在那里我熟悉了基础知识，他 继续建模，并设法创建一个数字复制智能城市。

是的，3D建模的知识对我非常有用，我是 肯定他们将来会有用。 但我的主要方向是 选择非常这并不难：我被提供把我的空闲时间 随着另一个圈子，我我决定选择的方向 编程。

我已经 写道 一篇关于在途中等待程序员的文章。 我愿意 祝你好运所有那些谁是刚刚开始出现在这个 方向，并继续为他学习一门新的编程语言。

当然，对所有其他人，我也想表达一个巨大的感谢 祝你一切顺利。 祝你好运！
